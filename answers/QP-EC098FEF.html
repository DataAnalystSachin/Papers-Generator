Q1: सिंटैक्स
सिंटैक्स प्रोग्रामिंग भाषा की मूल संरचना है, जो यह निर्धारित करता है कि कोड कैसे लिखा जाना चाहिए।

Q2: //
C, C++, और Java जैसी भाषाओं में एकल पंक्ति टिप्पणी के लिए '//' का उपयोग किया जाता है।

Q3: इंटीजर
इंटीजर डेटा प्रकार संख्यात्मक डेटा को दर्शाता है, जैसे 1, 2, 3 आदि।

Q4: def
Python में फ़ंक्शन को परिभाषित करने के लिए 'def' का उपयोग किया जाता है।

Q5: एक नामित स्थान
चर (variable) एक नामित स्थान है जहाँ डेटा को संग्रहीत किया जाता है और इसे प्रोग्राम में उपयोग किया जा सकता है।
