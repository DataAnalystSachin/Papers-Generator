Answer Key

Q1: .xlsx
The default file extension for an Excel workbook in recent versions is .xlsx.

Q2: SUM
The SUM function is used to add together a range of numbers in Excel.

Q3: Copies data to adjacent cells
The AutoFill feature allows users to quickly copy data or extend a series of numbers or dates to adjacent cells.

Q4: Image
While Excel can display images, 'Image' is not considered a valid data type for cells.

Q5: To keep specific rows or columns visible while scrolling
'Freeze Panes' allows users to keep certain rows or columns visible while scrolling through the rest of the worksheet.
